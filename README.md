# Industrial Data Quality Automation

## Contexte
Mémoire réalisé à Aivancity Paris.

Sujet : Optimisation et automatisation du contrôle qualité des données dans un système d’information industriel.

## Objectif
Automatiser les contrôles qualité afin d’améliorer la fiabilité des données et réduire les interventions manuelles.

## Stack technique
- Python (pandas, numpy)
- SQL
- Power BI
- Excel

## Résultats
- Réduction du temps de contrôle manuel
- Amélioration de la fiabilité des indicateurs
- Standardisation des règles métier

## Compétences démontrées
- Automatisation de processus data
- Contrôle qualité
- Conception de dashboards décisionnels
